# Sprint 3 IT Academy | Video Management Tool

## Project Description
This project corresponds to **Sprint 3 of the Frontend Bootcamp (IT Academy – Barcelona Activa)**.  
The main goal is to strengthen the use of **JavaScript array methods** and apply **unit testing with Jest** to ensure code quality.

---

## Technologies Used
- **JavaScript (ES6+)**
- **Node.js**
- **Jest** for testing
- **Git & GitHub**

---

## How to Clone and Run the Project

### 1. Clone the repository  
In your terminal, run:
```
git clone https://github.com/nataliarose98/S3-Testing.git
```

### 2. Enter the directory:

```
cd starter-code-frontend-sprint-3-movies
```

### 4. Install dependencies  
Make sure you have Node.js installed.  
Then run:
```
npm install
```

### 5. Run the tests
```
npm run test:watch
```
